# 管理画面仕様 (Admin UI Specification)

## 1. 概要
管理者がコンテンツのメタデータ（タイトル、説明文、公開状態など）を直接編集し、配信カタログを更新するためのWebインターフェース。

## 2. 主要機能
### 2.1 コンテンツ一覧 (Content List)
-   インポートされたすべてのコンテンツをグリッド/リスト形式で表示。
-   ステータス（公開/非公開）、タイプ（動画/360/画像）、作成日によるフィルタリング。

### 2.2 コンテンツ編集 (Content Editor)
-   **基本情報編集**:
    -   タイトル (Title)
    -   説明文 (Description) - リッチテキストまたはプレーンテキスト
    -   公開日時 (Published At)
-   **動画詳細**:
    -   監督 (Director)
    -   解像度 (Resolution)
-   **アセット管理**:
    -   サムネイル/ポスター画像のアップロード・変更。
    -   紐付けられているHLSパスの確認。

### 2.3 公開・同期 (Publish & Sync)
-   保存時に `Metadata Service`（PostgreSQL）を更新。
-   更新完了後、自動的に `Catalog Service`（MongoDB/Redis）へ即時同期（Online Sync）をトリガーする。

## 3. 画面設計（イメージ）
-   **Dashboard**: 直近インポートされた「未編集」のビデオを強調表示。
-   **Edit Page**: 左側にプレビュー画面（またはポスター画像）、右側にフォームを配置した直感的なレイアウト。

## 4. セキュリティ
-   `Auth Service` によるAdmin権限チェック。
-   `/admin/*` ルートへのアクセス制限。
