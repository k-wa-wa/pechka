# フロントエンド・UI仕様

本ドキュメントでは、一般ユーザー向けの動画プレーヤーおよび管理者向けの管理画面に関するUI/UX仕様を定義する。

---

## 1. 動画プレーヤー仕様 (Player UI)

ユーザーが動画コンテンツを視聴するためのプレーヤー画面 (`/videos/watch/[short_id]`) の仕様。

### 1.1 基本機能
- **HLSストリーミング再生**: `hls.js` を用いたAdaptive Bitrate Streaming。ネイティブHLSをサポートするブラウザ（Safari等）ではネイティブ再生をフォールバックとして使用する。
- **再生制御**:
  - 再生/一時停止
  - **スキップボタン**: Play/Pauseボタンの左右に、指定秒数（設定値）巻き戻し・早送りするボタンを配置。
  - シークバーによる任意位置へのシーク
  - 音量調整・ミュート
  - 全画面表示（フルスクリーン）

### 1.2 ジェスチャー・ショートカット機能

#### ダブルタップ（ダブルクリック）によるスキップ
スマートフォンやタブレットなどのタッチデバイス、およびPCでの操作性を向上させるため、画面の左右エリアでのダブルタップ検知によるスキップ機能を実装する。
- **操作方法**:
  - プレーヤー画面の **左半分** をダブルタップ（ダブルクリック）すると、指定秒数 **巻き戻す**。
  - プレーヤー画面の **右半分** をダブルタップ（ダブルクリック）すると、指定秒数 **早送り（スキップ）** する。
- **視覚的フィードバック（Ripple Effect）**:
  - スキップ操作が行われた際、対象エリア（左または右）に半透明の波紋状のエフェクトや、「10秒」などのインジケーターを一時的に表示させ、操作が認知できるようにする。

#### PC向けキーボードショートカット
- **Spaceキー**: 再生 / 一時停止のトグル
- **左矢印キー (ArrowLeft)**: 指定秒数（設定値）巻き戻す
- **右矢印キー (ArrowRight)**: 指定秒数（設定値）早送りする
- **上矢印キー (ArrowUp) / 下矢印キー (ArrowDown)**: 音量を上げる / 下げる
- **F / fキー**: 全画面表示（フルスクリーン）のトグル
- **M / mキー**: ミュート / ミュート解除のトグル

#### シングルタップとの競合回避
- 画面中央部や、コントロールUI（シークバーなど）以外の領域でのシングルタップは「再生 / 一時停止」のトグルとして機能する。
- ダブルタップが検知された場合は、再生停止トグルよりもスキップ操作が優先されるよう、イベントの発火順序を制御する。

### 1.3 プレーヤー設定（スキップ秒数のカスタマイズ）
- **設定内容**: ユーザーは自身の好みに応じて、スキップする秒数を変更できる（5秒, 10秒, 15秒, 30秒）。
- **保存先**: 端末ごとのローカルストレージ (`localStorage`) に保存し、ログイン状況にかかわらずデバイス（ブラウザ）固有の設定として扱う。
- **UI/UX**: プレーヤーのコントロールバーに「設定（歯車）」アイコンを配置し、ポップオーバー形式で選択可能とする。

### 1.4 その他
- **UIの自動非表示**: マウス操作やタップがない場合、数秒（3秒程度）でコントロールUIを非表示にし、没入感を高める。

## 2. 管理画面仕様 (Admin UI)

管理者がコンテンツのメタデータを編集し、カタログを更新するためのWebインターフェース。

### 2.1 主要機能
- **コンテンツ一覧**:
  - 全コンテンツをグリッド/リスト形式で表示。
  - ステータス（公開/非公開）、タイプ（動画/360/画像）、作成日によるフィルタリングが可能。
- **コンテンツ編集**:
  - **基本情報**: タイトル、説明文、公開日時の編集。
  - **メディア詳細**: 監督名、解像度の確認・編集。
  - **アセット管理**: サムネイル/ポスター画像のアップロード・変更、HLSパスの確認。
- **公開・同期**:
  - 保存時に `Metadata Service`（PostgreSQL）を更新。
  - ~~更新完了後、`Catalog Service`（MongoDB）への即時同期をトリガーする。~~
  - 更新後は **Benthos** による自動同期により、数秒〜数十秒以内にカタログへ反映される。

### 2.2 セキュリティ
- `Auth Service` によるAdmin権限の厳格なチェック。
- `/admin/*` のすべてのルートへのアクセスを制限。
