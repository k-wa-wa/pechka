# フロントエンド 動画プレーヤー仕様 (Frontend Player UI Specification)

## 1. 概要
ユーザーが動画コンテンツを視聴するためのプレーヤー画面 (`/videos/watch/[short_id]`) の仕様を定義する。

## 2. 基本機能
- **HLSストリーミング再生**: `hls.js`を用いたAdaptive Bitrate Streaming。ネイティブHLSをサポートするブラウザ（Safari等）ではネイティブ再生をフォールバックとして使用する。
- **再生制御**:
  - 再生/一時停止
  - **スキップボタン**: Play/Pauseボタンの左右に、指定秒数（設定値）巻き戻し・早送りするボタンを配置。
  - シークバーによる任意位置へのシーク
  - 音量調整・ミュート
  - 全画面表示（フルスクリーン）

## 3. ジェスチャー・ショートカット機能

### 3.1 ダブルタップ（ダブルクリック）によるスキップ
スマートフォンやタブレットなどのタッチデバイス、およびPCでの操作性を向上させるため、画面の左右エリアでのダブルタップ検知によるスキップ機能を実装する。
- **操作方法**:
  - プレーヤー画面の **左半分** をダブルタップ（ダブルクリック）すると、指定秒数 **巻き戻す**。
  - プレーヤー画面の **右半分** をダブルタップ（ダブルクリック）すると、指定秒数 **早送り（スキップ）** する。
- **視覚的フィードバック（Ripple Effect）**:
  - スキップ操作が行われた際、対象エリア（左または右）に半透明の波紋状のエフェクトや、「10秒」などのインジケーターを一時的に表示させ、操作が認知できるようにする。

### 3.2 PC向けキーボードショートカット (Keyboard Shortcuts)
PCブラウザでの直感的な操作を提供するため、以下のキーボードショートカットを実装する。
- **Spaceキー**: 再生 / 一時停止のトグル
- **左矢印キー (ArrowLeft)**: 指定秒数（設定値）巻き戻す
- **右矢印キー (ArrowRight)**: 指定秒数（設定値）早送りする
- **上矢印キー (ArrowUp) / 下矢印キー (ArrowDown)**: 音量を上げる / 下げる
- **F / fキー**: 全画面表示（フルスクリーン）のトグル
- **M / mキー**: ミュート / ミュート解除のトグル

### 3.3 シングルタップとの競合回避
- 画面中央部や、コントロールUI（シークバーなど）以外の領域でのシングルタップは「再生 / 一時停止」のトグルとして機能する。
- ダブルタップが検知された場合は、再生停止トグルよりもスキップ操作が優先されるか、ユーザー体験を損ねない範囲で両立させる（実装上はReactの `onDoubleClick` を利用しつつ、必要に応じて自前でのタップ間隔計算を実施する）。

## 4. プレーヤー設定（スキップ秒数のカスタマイズ）

### 4.1 設定内容
ユーザーは自身の好みに応じて、スキップする秒数を変更できる。
- **選択肢**: `5秒`, `10秒` (デフォルト), `15秒`, `30秒` など
- **保存先**: アカウント情報ではなく、端末ごとのローカルストレージ (`localStorage`) に保存し、ログイン状況にかかわらずデバイス（ブラウザ）固有の設定として扱う。

### 4.2 UI/UX
- プレーヤーのコントロールバー（右下など）に「設定（歯車）」アイコンを配置する。
- アイコンをクリックすると、スキップ秒数を選択できる小さなポップオーバーメニュー（Popover / Dropdown）を表示する。

## 5. その他の要件
- 非アクティブ時のUI非表示：マウス操作やタップがない場合、数秒（例: 3秒）でコントロールUIやマウスカーソルを非表示にし、没入感を高める。
